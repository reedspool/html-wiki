HTML Wiki

You're at the homepage of this wiki.

"HTML Wiki" is an experimental work-in-progress tool for building and running websites. The tool is equally at home as a dynamic web server or as a static site generator. There are two novelties. First is a HTML-based templating language. Second, a query language for dynamic data.

In the future the hope is that this site will be a dog-fooded website generated by the tool, a home for documentation about the tool, and a testing platform.
